Knowledge Base Permission Levels

Understanding what each KB access level can see:

  • FSR Can view articles marked as FSR (all users can see)
  • NOC Can view FSR + NOC articles (NOC team level)
  • Admin Can view all articles including Admin-only content

Note: Users with "Create" permission can create, edit, and delete any KB article. Superadmins automatically have create permission.
